# Mapa-de-Desempenho-El-trico-DEC-FEC-
Uma solução completa para extração, tratamento, análise espacial e visualização de dados de qualidade do fornecimento de energia elétrica nos municípios brasileiros, com base nos dados oficiais da **ANEEL**. O projeto identifica áreas críticas que descumprem as metas regulatórias de **DEC** e **FEC**
